Primeiro item de navegação

In this section, in order to make the image reponsive, forcing it to mantain a squared proportion using CSS, the solution presented by the article " How to Create a Responsive Square with CSS was applied. It is based on replacing the "height" property by the ":after" pseudo-element, in "square" class, with "padding-bottom", considering the padding is calculated with respect to its parent "width".
